These apps are hosted on a free server, so they take a while to load. Thank you for your patience.


Foam

This application is designed to search for and explore beers. If you want to look up beer stats or explore new beers, then this app is for you. This app uses Untappd's API, but users can also add beers and breweries to the database. Foam's database is built on SQL and utilizes the Knex.js library for its functionality. The front end of the app runs on handlebars and the backend runs on node.js/express.


Roomix

This application is designed to help people find roomates. It can be hard finding roomates, so I built this user friendly app as a solution. This app uses Auth0 for user verification and mongodb to create and maintain user profiles. This is a traditional M.E.R.N stack app that utilizes Redux for state management. There is also a custom chat function that I built using the mongoose library.
